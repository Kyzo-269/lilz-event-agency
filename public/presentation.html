📅
Planning
Gestion des postes de travail de l'équipe avec vue liste et calendrier semaine. Assignation, alertes avant prise de poste et déplacement mobile-first. CEO · Admin · Chef de Projet · Site Manager
📋 Vue Liste
  • Tous les postes triés par date puis heure
  • Indicateur "Actif maintenant" (heure en cours)
  • Indicateur alerte (démarre / finit dans 30 min)
  • Filtre par date
  • Carte par poste : nom, rôle, heure début/fin, notes
  • Couleur de la carte selon le rôle assigné
  • Boutons ← → pour déplacer d'un jour (iOS-safe)
  • Suppression du poste
📆 Vue Calendrier Semaine
  • 7 colonnes Lun → Dim
  • Navigation semaine par semaine (← →)
  • Postes affichés dans la colonne du jour
  • Déplacement ← → entre les colonnes
  • Badges compacts avec heure et poste
  • Colonnes sans overflow (compatible iPhone)
➕ Création de poste
  • Assignation à un membre de l'équipe
  • 12 types de postes prédéfinis : Entrée, Bar, Scène, Vestiaire, Parking, Coulisses, Sécurité, Accueil VIP, Technique Son, Technique Lumière, Coordination, Autre
  • Date, heure début, heure fin
  • Notes libres
  • Push notification envoyée à la personne assignée
🔔 Alertes & Notifications
  • Notification locale 30 min avant chaque prise de poste
  • Recalcul automatique à chaque chargement
  • Fonctionne via l'API Notification du navigateur
  • Notification push distante à l'assignation
  • Compatibilité iOS Safari PWA

Notes internes — Messagerie groupe
Fil de messages partagé par toute l'équipe. Texte, vocal, photos, mentions, réactions, épinglage et alertes urgentes.
✍️ Envoi de messages
  • Texte avec toggle urgent (🚨)
  • Message vocal : enregistrement micro avec timer
  • Support iOS (audio/mp4) et Android (audio/webm)
  • Photo : sélection galerie + prévisualisation avant envoi
  • Médias uploadés sur Supabase Storage
  • Réponse (reply) avec citation du message cible
  • Mentions @rôle avec autocomplete (11 rôles)
🔖 Gestion des messages
  • Messages épinglés affichés en haut en permanence
  • Filtre "Urgents uniquement"
  • Réactions emoji : 👍 ❤️ 😂 🔥 👏 😮 (toggle ajout/retrait)
  • Suppression d'un message
  • Mentions surlignées en bleu, URGENT en rouge
  • Scroll auto vers le bas
  • Push notification à toute l'équipe à chaque envoi
✉️
Messages privés — Chat 1:1
Conversations directes entre deux membres. Temps réel, médias, push, son de notification et polling de secours.
📬 Liste des conversations
  • Toutes les conversations avec dernier message
  • Badge non-lus (compteur rouge)
  • Heure relative ("5min", "2h", date)
  • Barre de recherche par nom
  • Section "Nouvelle conversation" : tous les membres non contactés
  • Aperçu : texte, 🎙️ Vocal ou 📷 Photo
💬 Conversation ouverte
  • Bulles séparées par labels jour (Aujourd'hui / Hier)
  • Indicateur lu/non lu avec timestamp
  • Envoi texte, image, vocal
  • Marquage auto en lus à l'ouverture
  • Son à réception (Web Audio API)
  • Notification navigateur si onglet en arrière-plan
  • Realtime + polling 3s (robustesse iOS)
  • Push envoyé à l'interlocuteur

Infrastructure & Fonctions transversales
Socle technique de l'application : PWA, notifications push, temps réel, thème, authentification et sécurité par rôle.
📱 PWA & Mobile
  • Installable sur l'écran d'accueil iPhone / Android
  • Service Worker v2 avec cache réseau-first
  • Page hors-ligne personnalisée
  • Icônes iOS / Android adaptées
  • Compatible Safari iOS 16.4+
  • Interface safe-area (encoche, barre d'accueil)
🔔 Notifications Push
  • Standard VAPID web-push
  • Abonnement depuis le Dashboard
  • Activation iOS-safe (gesture context)
  • Réception même app fermée
  • Envoi via API sécurisée (JWT)
  • Stockage dans Supabase (RLS)
⚡ Temps réel & Sync
  • Supabase Realtime (postgres_changes)
  • Polling fallback toutes les 3s (iOS)
  • Optimistic updates + rollback
  • Scroll automatique vers les nouveaux messages
  • Rafraîchissement périodique (équipe, alertes)
🔒 Sécurité & Accès
  • Authentification Supabase Auth (email/password)
  • Row Level Security (RLS) sur toutes les tables
  • Accès par rôle sur chaque module
  • Module Finances : accès restreint avec page de blocage
  • JWT Authorization header (robustesse API routes)
  • Sessions persistantes (refresh token)
🎨 Interface
  • Thème clair / sombre global persisté
  • Couleurs comoriennes : vert, rouge, blanc, bleu, or
  • Bande décorative tricolore sur toutes les pages
  • Badges rôles colorés cohérents dans toute l'app
  • BottomNav persistante (navigation bas d'écran)
  • Animations d'entrée sur les listes
